# React Currency Converter
## Инструкция по запуску

Для установки и запуска приложения необходимо ввести след. команды:
```
npm install
cd react-currency-converter
npm start
```
Если используется менеджер пакетов `yarn`, то:
```
yarn install
cd react-currency-converter
yarn start
```
